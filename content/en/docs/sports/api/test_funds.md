To access `PLAY_EUR` for simulated betting, complete the following steps:

Meet the minimum balance requirement — Ensure your account balance is at least 10 EUR equivalent. This can be held in any supported currency such as BTC, ETH, USDC, USDT, etc.

Generate an API key — Create an API key from your account settings.

Contact customer support — Once your balance requirement is met and your API key is generated, contact customer support to request Play EUR funds.

Customer support will then credit the account with `PLAY_EUR`, which can be used to test betting strategies without risking real money.
